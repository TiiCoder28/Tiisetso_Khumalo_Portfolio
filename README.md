# Tiisetso_Khumalo_Portfolio
Get to know more about me in a fun and visually appealing portfolio created using Vue.js, Vite and Three.js. I had fun with creating the visuals for this portfolio, I hope you have fun exploring it and learning more about me too 🎉🌟

## Documentation

Project documentation is stored in the `docs/` folder.

- `docs/README.md` explains how documentation is organized.
- `docs/changes/` contains feature-by-feature delivery notes covering what changed, why it changed, and what outcome was achieved.
- `CONTENT_SYSTEM_GUIDE.md` is the system-level guide for the content platform, Sanity workflow, and Supabase retrieval layer.
